# Реализация модуля регистрации и аутентификации
## Приложение разработано для мобильных платформ с использованием языка Java. Представленная функциональность является частью учебного проекта, направленного на освоение ключевых механизмов.

Модуль регистрации пользователя включает:

Валидацию вводимых данных

* Подтверждение пароля (сравнение двух полей ввода)
* Активацию кнопки регистрации только при выполнении всех валидационных условий
* Автоматический переход на экран аутентификации после успешной регистрации

Модуль аутентификации обеспечивает:
*  Проверку существования учетной записи и корректности учетных данных
*  Функцию восстановления пароля через механизм верификации по одноразовому коду
